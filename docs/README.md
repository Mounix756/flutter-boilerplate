# Documentation

Guides conservés pour le noyau boilerplate:

- `ENVIRONMENT_BUILD.md`: configuration par environnement et builds.
- `LOCALIZATION.md`: langues, clés de traduction et ajout de langue.
- `NAVIGATION_FLOW.md`: flux splash, onboarding, auth, invité et app shell.
- `PUSH_NOTIFICATION.md`: notifications push et centre de notifications.
- `THEME_GUIDE.md`: thème clair/sombre/système.
- `WIDGETS_GUIDE.md`: composants UI partagés.
- `CONTRIBUTING.md`: conventions de contribution.

Les anciens documents produit ont été retirés pour garder le dépôt générique.
